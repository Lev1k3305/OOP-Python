# Rick & Morty Explorer (PyQt6) — Практическая работа №7

Графическое приложение для исследования вселенной Rick and Morty с использованием публичного API.

## Функционал
- Поиск персонажей, эпизодов, локаций
- Случайный персонаж
- Список всех эпизодов с фильтром по сезону
- Энциклопедия избранных персонажей (с рейтингом 1–5)
- История действий
- Красивый неоново-зелёный стиль в духе портала

## Установка и запуск
```bash
pip install PyQt6 requests
python main.py
